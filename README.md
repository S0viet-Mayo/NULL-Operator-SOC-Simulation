# SOC attack Simulation Game 

## Overview
This project is a text-based choice based SOC (Security Operations Center) simulator in which player choices will affect the outcome of the game. Real world attack patterns/stages have been used to model the game as closley to real life SOC operations as possible. There are 5 possible endings with one secret ending. 

The goal of this project is to demonstrate how cyber attacks evolve from initial detection to full system compromise and how defensive devisions impact outcomes at each stage. 

## Research Question
How do IoT botnet infections, such as Mirai-style botnets, propagate across multiple devices, and what defensive signals can detect each stage of their spread?

## Features 
- Branching decision-based gameplay
- Simulated SOC alert systems
- Realistic attack progress modeled after IoT Botnet behavior, specifically Mirai-style attacks.
- Multiple endings based on response quality of player
- Incident-style explanations for cybersecurity concepts. 

## Educational Focus
This project was created for an UnEssay project to present my research in a fun and interactive way. While this simulator is good fun, it follows real world attack patterns and user choices follow real-world choices!
